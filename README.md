Идея
Multi-tenant EdTech-платформа для детей (7–15 лет).
Ядро - Teacher Studio (создание уроков), совмещённое с игровой системой обучения (CodeQuest-подход).

Цель
- Учитель создаёт и публикует урок
- Ученик проходит интерактивную сессию (код / блоки)
- Родитель контролирует доступ и прогресс
- Платформа анализирует обучение и выдаёт статистику


Ключевая особенность продукта
- Плавный переход:
  Blockly (7–10) -> гибрид -> текстовый код (Python)
- Обучение через игру:
  XP, уровни, достижения, streak, задания
- Не просто LMS, а интерактивная среда программирования


Что уже реализовано

Backend
- Django + DRF + JWT
- Celery + Redis
- PostgreSQL

Frontend
- Vue 3 + TypeScript
- SPA архитектура

Дополнительно
